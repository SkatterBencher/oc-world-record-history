About This Museum

The OC World Record Museum is a curated, open archive of CPU, GPU, and memory overclocking world records dating back to 1996. Every entry is sourced, verified, and preserved for posterity.

Methodology

Records are verified against contemporary sources — CPU-Z validations, HWBot entries, forum posts, and archived pages. Where the original source URL has died, we maintain a local archive via ArchiveBox.

Contributing

Found an error or a missing record? Open an GitHub Issue to discuss it, or submit a Pull Request following the contributor guidelines .
